reRunReports {ytBangerz}R Documentation

Re-run Reports

Description

Re-run Reports

Usage

reRunReports(date, sites = NULL)

Arguments

date

A YYYY-MM-DD character date. Input a value to re-run the report generation for that date.

sites

Optional, defaults to NULL. If NULL all sites are re-run. Please, no OKSR.

Value

Report files aggreagted up from site level stats, by date.
