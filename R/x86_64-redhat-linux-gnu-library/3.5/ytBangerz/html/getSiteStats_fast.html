Get Site Statisitcs, The Fast Way

Description

Get Site Statisitcs, The Fast Way

Usage

getSiteStats_fast(site, l0.matrix, non.1hz.dp, dates, creds, range.table,
  raw.stats.dir, exception.df)

Arguments

l0.matrix

The l0.matrix as a data frame.

non.1hz.dp

A data frame of lookup/convert information for non-1Hz data products

dates

A list of startDate and endDate (exclusive) to summarize over. Generated by a call to ytBangerz:::define.dates()

creds

Presto creds, in the form "user;pass". Use NULL if no creds to use the SOM VM's system creds

range.table

A data frame of sensor/stream ranges

raw.stats.dir

The directory where stat files are saved

exception.df

A data frame of excepted streams. Full 45-character streams and site column required

sites

A list of sites in 4-Letter NEON site code format

Value

Statistic files for each site are saved to the specified directory
