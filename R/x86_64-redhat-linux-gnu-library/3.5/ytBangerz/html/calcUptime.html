Calculate Uptime

Description

Calculate Uptime

Usage

calcUptime(l0.data, non.1hz.dp, streams, startDate, endDate)

Arguments

l0.data

a data frame of 1) timestamps, 2) stream IDs, and 3) values

non.1hz.dp

A data frame summarizing non-1Hz streams and their approprate conversion factors

streams

A list of streams expected at the site

startDate

A POSIXct object of the start date/time for the query - used to check that all data were pulled.

endDate

A POSIXct object of the start date/time for the query (exclusive) - used to check that all data were pulled.

Details

From a set of L0 data, calculate uptime for all expected streams.

Value

A list of Uptimes corresponding to the input streams
