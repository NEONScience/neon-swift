makeStatusDf {ytBangerz}R Documentation

Make Status Dataframe

Description

This function generates a matrix from condition and dp.id

Usage

makeStatusDf(dp.id, site, l0.matrix, conditions)

Arguments

dp.id

character vector of full NEON Data Product Identifiers, e.g., "NEON.D10.CPER.DP0.00004.001.00474.000.035.000"

site

character class defining site of interest, e.g., "CPER"

l0.matrix

matrix of L0 data products

conditions

???

Value

a site-specific dataframe comprising uptimes of data product statuses
