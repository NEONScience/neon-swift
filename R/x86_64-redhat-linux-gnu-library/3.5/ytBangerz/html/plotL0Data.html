plotL0Data {ytBangerz}R Documentation

Plot problematic L0 streams

Description

This function plots L0 data for quick analyses within the domain reports

Usage

plotL0Data(site, stream, date, creds = NULL, rangeTable)

Arguments

site

character class defining site of interest, e.g., "CPER"

stream

character vector of full NEON Data Product Identifiers, e.g., "NEON.D10.CPER.DP0.00004.001.00474.000.035.000"

creds

character class defining username and password of PDR credentials for data access

rangeTable

class data table defining the upper and lower ranges of the expected data; used for green shading

Value

html plot for domain report
