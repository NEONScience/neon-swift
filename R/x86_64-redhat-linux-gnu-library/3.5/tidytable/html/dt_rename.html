dt_rename {tidytable}R Documentation

Rename variables by name

Description

Rename variables from a data.table.

Usage

dt_rename(.data, ...)

Arguments

.data

A data.frame or data.table

...

Rename expression like dplyr::rename()

Value

A data.table

Examples

dt <- data.table::data.table(x = c(1,2,3), y = c(4,5,6))

dt %>%
  dt_rename(new_x = x,
            new_y = y)
