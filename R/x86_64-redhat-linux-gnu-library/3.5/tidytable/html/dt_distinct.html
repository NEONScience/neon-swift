dt_distinct {tidytable}R Documentation

Select distinct/unique rows

Description

Retain only unique/distinct rows from an input df.

Supports enhanced selection if dots are used

Usage

dt_distinct(.data, ...)

Arguments

.data

A data.frame or data.table

...

Columns to select before determining uniqueness. If omitted, will use all columns

Value

A data.table

Examples

example_dt <- data.table::data.table(
  x = 1:3,
  y = 4:6,
  z = c("a", "a", "b"))

example_dt %>%
  dt_distinct()

example_dt %>%
  dt_distinct(z)
