dt_count {tidytable}R Documentation

Count observations by group

Description

Returns row counts of the dataset. If bare column names are provided, dt_count() returns counts by group.

Supports enhanced selection

Usage

dt_count(.data, ...)

Arguments

.data

A data.frame or data.table

...

Optional: Bare column names to group by

Value

A data.table

Examples

example_df <- data.table::data.table(
  x = 1:3,
  y = 4:6,
  z = c("a", "a", "b"))

example_df %>%
  dt_count()

example_df %>%
  dt_count(z)

example_df %>%
  dt_count(is.character)
