dt_group_split {tidytable}R Documentation

Split data frame by groups

Description

Split data frame by groups. Returns a list

Usage

dt_group_split(.data, ...)

Arguments

.data

A data.frame or data.table

...

Groups to split by

Examples

test_df <- data.table::data.table(
  a = 1:5,
  b = 1:5,
  c = c("a","a","a","b","b"),
  d = c("a","a","a","b","b"))

test_df %>%
  dt_group_split(c, d)
