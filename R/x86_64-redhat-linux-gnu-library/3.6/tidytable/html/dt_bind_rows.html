dt_bind_rows {tidytable}R Documentation

Bind data.tables by row and column

Description

Bind rows or columns together.

Usage

dt_bind_rows(.data, ..., .id = NULL)

dt_bind_cols(.data, ...)

Arguments

.data

A data.table to bind, or a list of data.tables

...

Data frames to bind

.id

If TRUE, an integer column is made as a group id

Examples

df1 <- data.table::data.table(x = c(1,2,3), y = c(3,4,5))
df2 <- data.table::data.table(x = c(1,2,3), y = c(3,4,5))

df1 %>%
  dt_bind_rows(df2)

dt_bind_rows(list(df1, df2))

df1 %>%
  dt_bind_cols(df2)

dt_bind_cols(list(df1, df2))
