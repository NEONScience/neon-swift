dt_starts_with {tidytable}R Documentation

Select helpers

Description

These functions allow you to select variables based on their names.

  • dt_starts_with(): Starts with a prefix

  • dt_starts_with(): Ends with a suffix

  • dt_contains(): Contains a literal string

  • dt_everything(): Matches all variables

Usage

dt_starts_with(match)

dt_contains(match)

dt_ends_with(match)

dt_everything()

Arguments

match

The match for the helper function to use

Examples

example_dt <- data.table::data.table(
  x = 1,
  y = 2,
  double_x = 2,
  double_y = 4)

example_dt %>%
  dt_select(dt_starts_with("x"))

example_dt %>%
  dt_select(dt_ends_with("y"))
