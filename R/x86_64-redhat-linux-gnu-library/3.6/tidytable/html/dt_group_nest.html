dt_group_nest {tidytable}R Documentation

Nest data.tables

Description

Nest data.tables by group

Supports enhanced selection

Usage

dt_group_nest(.data, ..., .key = "data")

Arguments

.data

A data.frame or data.table

...

bare column names to group by. If empty nests the entire data.table

.key

Name of the new column created by nesting

Examples

test_df <- data.table::data.table(a = 1:10,
  b = 11:20,
  c = c(rep("a", 6), rep("b", 4)),
  d = c(rep("a", 4), rep("b", 6)))

test_df %>%
  dt_group_nest()

test_df %>%
  dt_group_nest(c, d)

test_df %>%
  dt_group_nest(is.character)
