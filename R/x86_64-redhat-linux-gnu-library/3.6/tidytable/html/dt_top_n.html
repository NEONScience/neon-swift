dt_top_n {tidytable}R Documentation

Select top (or bottom) n rows (by value)

Description

This is a wrapper that uses dt[order()] to select the top or bottom entries in each group, ordered by wt.

Usage

dt_top_n(.data, n = 5, wt = NULL, by = NULL)

Arguments

.data

A data.frame or data.table

n

Number of rows to return

wt

(Optional). The variable to use for ordering. If NULL uses the last column in the data.table.

by

list() of bare column names to group by

Value

A data.table

Examples

example_dt <- data.table::data.table(
  x = 1:5,
  y = 6:10,
  z = c(rep("a", 3), rep("b", 2)))

example_dt %>%
  dt_top_n(2, wt = y)

example_dt %>%
  dt_top_n(2, wt = y, by = z)
