dt_unnest_legacy {tidytable}R Documentation

Unnest a nested data.table

Description

Unnest a nested data.table.

Usage

dt_unnest_legacy(.data, col)

Arguments

.data

A nested data.table

col

The column to unnest

Examples

nested_df <- data.table::data.table(
  a = 1:10,
  b = 11:20,
  c = c(rep("a", 6), rep("b", 4)),
  d = c(rep("a", 4), rep("b", 6))) %>%
  dt_group_nest(c, d)

nested_df %>%
  dt_unnest_legacy(data)

nested_df %>%
  dt_unnest_legacy(data)
