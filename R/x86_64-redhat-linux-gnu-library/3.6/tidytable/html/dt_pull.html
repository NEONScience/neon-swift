dt_pull {tidytable}R Documentation

Pull out a single variable

Description

Pull a single variable from a data.table as a vector.

Usage

dt_pull(.data, var = NULL)

Arguments

.data

A data.frame or data.table

var

The column to pull from the data.table

Examples

example_dt <- data.table::data.table(
  x = c(1,2,3),
  y = c(4,5,6))

example_dt %>%
  dt_pull(y)
