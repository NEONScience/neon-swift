dt_mutate_if {tidytable}R Documentation

Mutate variants

Description

Edit existing columns. Supports enhanced selection.

There are two variants:

  • dt_mutate_all()

  • dt_mutate_across(): Replaces both mutate_if() & mutate_at()

Usage

dt_mutate_if(.data, .predicate, .funs, ..., by = NULL)

dt_mutate_at(.data, .vars, .funs, ..., by = NULL)

dt_mutate_across(.data, .cols, .funs, ..., by = NULL)

dt_mutate_all(.data, .funs, ..., by = NULL)

Arguments

.data

A data.frame or data.table

.predicate

predicate for dt_mutate_if() to use

.funs

Functions to pass. Can pass a list of functions.

...

Other arguments for the passed function

by

A single unquoted column or a list() of columns to group by.

.vars

vector c() of bare column names for dt_mutate_at() to use

.cols

vector c() of bare column names for dt_mutate_across() to use

Examples

library(data.table)

example_dt <- data.table::data.table(
  x = c(1,1,1),
  y = c(2,2,2),
  z = c("a", "a", "b"))

example_dt %>%
  dt_mutate_across(is.numeric, as.character)

example_dt %>%
  dt_mutate_across(c(x, y), ~ .x * 2)

example_dt %>%
  dt_mutate_across(c(x, y), list(new = ~ .x * 2))
