sendGroupedEmail {ytBangerz}R Documentation

Send grouped emails

Description

Function to send group-based (e.g., TIS sensors, AIS sensors, etc.) to recipients base on their request

Usage

sendGroupedEmail(email, sensorGroup, domain, domainReportText)

Arguments

email

character vector comprising recipient's email

sensorGroup

character class defining sensor group, e.g., "tis"

domain

Character class, domain of interest for the email

domainReportText

Text output for given date, to be subset to specific domain- and sensor-group for email recipients

Value

html email subset to domain and sensor group
