makeAlertOutput {ytBangerz}R Documentation

Make Alert Output

Description

This function formats the alert output

Usage

makeAlertOutput(domain.report.text, date, systemName = NULL)

Arguments

domain.report.text

text output of human readable sensor-related issues by domain

date

POSIX class denoting date of report to be run

systemName

Logical class defining separate class for domain reports

Value

html (?) file of domain alerts for a given day
