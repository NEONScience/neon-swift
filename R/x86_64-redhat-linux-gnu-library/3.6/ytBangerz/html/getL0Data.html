Description

Get L0 Data. All of it. Very Slow.

Usage

getL0Data(site, streams, startDate, endDate, creds)

Arguments

site

The 4-Letter site code for a NEON site. Please, no OKSR.

streams

A list of streams to pull data for.

startDate

A POSIXct object of the start date/time for the query

endDate

A POSIXct object of the start date/time for the query (exclusive)

creds

Presto creds, in the form "user;pass". Use NULL if no creds to use the SOM VM's system creds

Details

A wrapper function to handle dates and creds correctly. Not much to see. Real action is in prestoQuery!

Value

Site uptime stats are normalized (as appropriate) and written into the site's statistic files.
