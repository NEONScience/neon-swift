rangeFromPercentile {ytBangerz}R Documentation

Distill Percent-in-range Information from Bulk Statistics

Description

This function returns Percent-in-range values for streams

Usage

rangeFromPercentile(site.stats, range.table)

Arguments

site.stats

a data frame of site statistics, generated by getSiteStats_fast

range

class dataframe, contaning acceptable range values for streams

Value

The bulk statsitics data frame with an percent.in.range column
